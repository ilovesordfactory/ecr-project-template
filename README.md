# luau-ecr-template
A project template for luau projects that use the ecr library, it also has vide

## Getting Started
First you will need lune then,
Run these commands:

Feel free to change anything

```bash
lune run scripts/install-packages.luau
lune run scripts/dev.luau
```

## What is this?
A project template for ecr that adds basic replication along with personal player data replication.
Easily setup darklua, lune scripts, and wally package development.
